# Privacy Policy

## English

### 1. Data Collection
AppMusica is a fully offline, self-contained application. We **do not** collect, store, transmit, or sell any personal data. There are no tracking scripts, analytics, or telemetry of any kind embedded in this software.

### 2. Local Storage
All data generated by the application, including download queues, custom settings, UI preferences, and cookie files, are stored **strictly locally** on your device (within your user profile directory or the application folder). We have no access to this data.

### 3. Third-Party Interactions
When you use AppMusica to download media or search for content, the application communicates directly with third-party public APIs (e.g., YouTube, Apple Music, Shazam). 
- Your IP address is exposed to these services exactly as it would be if you visited their websites in a standard web browser.
- If you choose to load a `cookies.txt` file to bypass age-restrictions or premium barriers, these cookies are sent directly to the respective platform. We do not intercept or store your login credentials.

### 4. Open Source Transparency
You are free to inspect the source code of this application at any time to verify the absence of data collection mechanisms.

## Português (Brasil)

### 1. Coleta de Dados
O AppMusica é um aplicativo totalmente independente. Nós **não** coletamos, armazenamos, transmitimos ou vendemos nenhum dado pessoal. Não existem scripts de rastreamento, análises de uso (analytics) ou telemetria de qualquer tipo embutidos neste software.

### 2. Armazenamento Local
Todos os dados gerados pelo aplicativo, incluindo filas de download, configurações, preferências de interface e arquivos de cookies, são armazenados **estritamente de forma local** no seu dispositivo. O desenvolvedor não tem nenhum tipo de acesso a esses dados.

### 3. Interações com Terceiros
Quando você usa o AppMusica para baixar mídia, o aplicativo se comunica diretamente com APIs públicas de terceiros (ex: YouTube, Apple Music, Shazam).
- O seu endereço de IP é exposto a esses serviços exatamente da mesma forma que seria se você os acessasse através de um navegador comum.
- Se você carregar um arquivo `cookies.txt` para contornar restrições, esses cookies são enviados diretamente do seu PC para o YouTube/Spotify. Nós não interceptamos, lemos ou capturamos suas senhas.

### 4. Transparência 
Por ser um software de código aberto (Open Source), qualquer pessoa é livre para inspecionar o código deste aplicativo a qualquer momento para comprovar a total ausência de mecanismos de espionagem.
